Admin Approval

Aktivasi user tanpa buka dashboard Supabase. Buka dari HP, paste User ID, masukkan PIN, lalu klik Approve.

Tip: link dari WhatsApp bisa auto-isi User ID.

PIN disimpan di HP kamu (localStorage) biar next tinggal klik.

Pastikan konfigurasi sudah benar:
• Set SUPABASE_URL & ANON_KEY di public/js/supabaseClient.js
• Deploy Supabase Edge Function: approve_user
